template<typename char_type >
bool vinks::core::charset::basic_source::is_representable_as_control_char ( char_type  c)

Checks if passed character is a Control characters in basic source character set.

Parameters
cCharacter to be tested
Returns
  • true, if c is representationally equivalent to one of basic_source::control_char_type
  • false, otherwise
Precondition
  • char_type is assumed to be a type equivalent to a C++ character type.
  • object of char_type is said to have a value which is convertible to basic_source::base_type
Remarks
An object val of char_type is said to be representationally equivalent to basic_source::control_char_type if oand only if val is bitwise equivalent to ASCII value represented by one of basic_source::control_char_type
